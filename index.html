What is Machine Learning?

First, some Jargon

  • Artificial Intelligence: Machine mimics "cognitive" functions that humans associate with other human minds, such as "learning" and "problem solving"
  • Machine Learning: General process of dynamically modifying algorithm/function
  • Neural Network: A type of machine learning using simulated connected "neurons"
  • Artificial Intelligence is the GOAL
  • Machine Learing is the REQUIREMENT
  • Neural Network is an IMPLEMENTATION

Machine Learning is the requirement that must be met while pursuing AI.

What problems can it solve?

  • Classification: assign probabilities to certain classes
  • Regression: predicting a final value
  • Clustering: grouping based on similarity

How do we get the right weights and biases / thresholds?

Training Neural Networks

Just like how humans need lots of training to learn, we need to find a way to update the weights and biases as we train the neural network

Biological Neural Net Training

  • Multiple Dynamic Sensory Inputs
  • Dynamic Brain Chemistry / Neuro Transmitters / Hormones
  • Plasticity
  • Sleep Cycles / Memory Consolidation / Pruning / Sync

Artificial Neural Net Training

  • Minimize a Cost/Error Function with Math

What is a Cost Function?

  • Mean Squared Error ( Linear Regression )
  • Cross Entropy
  • Kullback-Leibler (KL) Divergence
    • http://www.machinephilosopher.com/cost-function-neural-network-intro/

Enter Stochastic Gradient Descent (Not Batch)

  • Allows multi-dimensional cost/error minimization
  • Uses Calculus Chain Rule
  • Propagates error corrections through the network
  • REQUIRES a differentiable activation function
  • Learning Rate CRUCIAL for convergence
  • Stochastic Gradient Descent (SGD) makes large training sets scalable and helps avoid fixating on local minimums
  • Mini Batch training can be used for large neural nets
  • Techniques like adding momentum can help reach global min...
http://blog.datumbox.com/tuning-the-learning-rate-in-gradient-descent/

Gradient Descent Explained (see links)

https://www.youtube.com/watch?v=eikJboPQDT0 https://www.youtube.com/watch?v=Fn8qXpIcdnI

Summary of Learning Process (Supervised)

  1. "Random" weights and biases/thresholds generated
  2. Input values applied -> creates output (don't forget to randomize)
  3. Cost function calculates error from output vs expected output
  4. Backpropagation (stochastic gradient descent) updates biases/thresholds (repeat 2 - 4 until cost converges)

Did it work?

  • You can report your accuracy against your training data
  • You can report out convergence of cost function
  • You MUST run additional TEST data through to verify a good general fit
  • When gathering data, remember you will need some for training, some for verifying

Did it not work?

Things you can adjust

  • Training Data Ammount
  • Training Data Repeats (epoch)
  • Address possible overfitting (dropout)
  • Less random initial weights and biases
  • Different activation function
  • Switch batch size
  • Modify structure of your net
  • Your self esteem

Deep Learning &
Network Types

Deep Learning

Deep learning is a badge of honor. The more hidden layers.. the more LEGIT you are!

Deep learning is relative and means that you have at least more than one hidden layer.

Why DL Works? (Expressibility)

  • Single perceptron layer proven to be limited to linear classification
  • What class of functions can a neural net express?
    • Composable non-linear layers can approximate any polynomial
    • Stone–Weierstrass theorem
    • Gives us all continuous functions on a closed interval for free!
https://www.youtube.com/watch?v=5MdSE-N0bxs

Why DL Works? (Efficiency)

  • Phase Space is significantly reduced by our low entropy world
  • How many possible images in a 1000x1000 black and white image?
  • 2^1,000,000
  • Particles in our Universe?
  • 10^89
  • Linearly Collapsable Models can be More Efficient in Deeper Representations
  • Many complex systems are modeled using low degree polynomials
  • Homework: Watch Max Tegmark

Common Types of Neural Nets

  • Feedforward neural network (this presentation)
  • Radial basis function (RBF) network
  • Kohonen self-organizing network.
  • Recurrent neural network (sequences)
https://en.wikipedia.org/wiki/Types_of_artificial_neural_networks

Neural Net Layer Structures

  • Fully Connected Layers
