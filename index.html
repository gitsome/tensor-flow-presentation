What is Machine Learning?

Jargon

  • Artificial Intelligence: Machine mimics "cognitive" functions that humans associate with other human minds, such as "learning" and "problem solving".
  • Machine Learing: General process of dynamically modifying algorithm/function
  • Deep Learning: A type of machine learning using multi-layer neural networks
  • Artificial Intelligence is the GOAL
  • Machine Learing is the REQUIREMENT
  • Deep Learning is an IMPLEMENTATION
