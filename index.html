What is Machine Learning?

First, some Jargon

  • Artificial Intelligence: Machine mimics "cognitive" functions that humans associate with other human minds, such as "learning" and "problem solving"
  • Machine Learning: General process of dynamically modifying algorithm/function
  • Neural Network: A type of machine learning using simulated connected "neurons"
  • Artificial Intelligence is the GOAL
  • Machine Learing is the REQUIREMENT
  • Neural Network is an IMPLEMENTATION

Machine Learning is the requirement that must be met while pursuing AI.

What problems can it solve?

  • Classification: assign probabilities to certain classes
  • Regression: predicting a final value
  • Clustering: grouping based on similarity

Why Even Try Neural Networks?

  • Pattern Recognition is a powerful tool
  • Brains are exceptional at pattern recongition and learning
  • Imagine the conditional statements you would have to write to detect a cat in a video!

What is a Neural Network?

It is just a function!

f([i1, i2, i3]) -> [o1, o2]

Classification Function

Classify black and white 10x10 images into 1 of 4 labels

f([i1, i2, i3, ..., i100]) -> [o1, o2, o3, o4]

Regression Function

Use 20 features of homes to predict a home value

f([i1, i2, i3, ..., i20]) -> [o1]
